Orderbook storage.

## Description

SFXDX Back-End Developer test assignment.

## Installation

```bash
$ npm install
```

You can set up your MongoDB connection along with your wallet info and Infura keys in
```
src/resources/config/constants.ts
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

API docs can be found here after running the server: http://localhost:3000/api

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
